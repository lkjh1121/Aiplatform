RDBMS(Relational Database Management System) 
   : 관계형데이터베이스시스템 
   관계 : 데이터와 데이터의 관계성 차원에서 바라보는 관점 
   데이터를 행과 열로 나누어서 테이블 구조로 바라본다(정형화 되어 있다)
   테이블과 테이블 관계로 데이터를 저장하는 형태의 데이터베이스이다. 
   개발자가 어떤 언어로 개발을 하던 원본이 되는 데이터는 별도의 저장공간에
   별도의 언어로 저장되어져서 실제 개발언어와 상관없이 데이터 관리가 
   되어야 한다 

   SQL(Query)-질의어 : 데이터를 데이터베이스에 추가 또는 수정 삭제, 
   조회등을 도와주는 언어 

   오라클(오라클사가 만든다) : 사용할때마다 라이센스를 줘야 한다 
                             대용량, 고속, 
                             대부분의 기업이나 국가기관

   MySQL -> Mysql 자체는 오라클사에 팔여서 오라클사가 만들고 
            유료로 바뀌었음 
            이전에 mysql  만들던 분들이 회사 차려서 mariadb 를 
            만들고 있는데 거의 흡사하다 

   빅데이터(몇천만건, 데이터가 보관될 필요가 없는 데이터)
